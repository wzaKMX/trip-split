# TripSplit — разделение долгов в поездках

Веб-приложение для учёта общих трат в поездках (аналог Splitwise) с двумя ускорителями ввода:

- 🎤 **Голосовой ввод** — скажите «я заплатил 1500 за такси на всех», и трата распознаётся и предзаполняется автоматически.
- 📷 **Чеки** — сфотографируйте чек, сумма/дата/позиции извлекаются автоматически.

Плюс обычный ручной ввод, расчёт балансов и подсказка минимальных переводов («кто кому должен»).

## Стек

- **Next.js 16** (App Router) + TypeScript + Tailwind CSS, mobile-first.
- **Dexie (IndexedDB)** — данные (включая фото чеков) хранятся локально в браузере.
- **Claude API** (`@anthropic-ai/sdk`, модель `claude-haiku-4-5`) — парсинг голоса и распознавание чеков через серверные API-роуты Next.js (ключ не покидает сервер).
- **Web Speech API** — распознавание речи прямо в браузере.

## Запуск

```bash
npm install
cp .env.local.example .env.local   # и вставьте свой ANTHROPIC_API_KEY
npm run dev
```

Откройте http://localhost:3000.

> Без `ANTHROPIC_API_KEY` приложение полностью работает в ручном режиме; голос и чеки покажут понятную ошибку.
> Голосовой ввод требует браузера с поддержкой Web Speech API (Chrome/Edge/Safari) и доступа к микрофону.

## Как пользоваться

1. Создайте поездку и добавьте участников. Выберите «Это я» — чтобы голосовое «я заплатил» понималось правильно.
2. Нажмите «+ Добавить трату»: голос, чек или вручную. Голос и чек предзаполняют форму — остаётся проверить и сохранить.
3. Вкладка «Кто кому должен» показывает балансы и минимальный список переводов.

## Структура

- `src/lib/` — типы, Dexie-БД (`db.ts`), расчёт балансов и переводов (`settle.ts`), клиент Anthropic.
- `src/app/api/parse-expense` и `parse-receipt` — серверные роуты с Claude (tool use / vision).
- `src/components/` — UI: форма траты, голос, чек, список, балансы, переводы.
- `src/hooks/useTripData.ts` — реактивное чтение из IndexedDB.
